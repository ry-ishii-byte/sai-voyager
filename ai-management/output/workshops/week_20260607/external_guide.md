## ファシリテーションのポイント

### アイスブレイク（「エシカル探偵」）

- 「良いアイデアの裏側を探す」という設定が探偵らしくて小中学生に響く。
- 「最近聞いた面白いビジネス」が浮かばない生徒には：
  - 「Uberみたいな配達サービス」「フリマアプリ」「YouTuberが稼ぐ仕組み」など具体例を提示する。
- このアイスブレイクで「良いアイデアにも問題がある」という問題意識の種を植えることが目的。答えを出させようとしない。
- チームが「問題なし」で止まったら「100万人が使ったとしたら？」と視野を広げる問いを出す。

### 導入講義（3つの柱フレームワーク）

- 「倫理」「持続可能」は日本語として難しいため、常に平易な言い換えを用意する：
  - 倫理 → 「公平か」「正直か」「誰かが不当に傷つかないか」
  - 持続可能 → 「ずっと続けられるか」「地球が受け入れられるか」「未来の人たちも使えるか」
- スライド6の「サステナビリティの三角形」は板書でも描いておくと、グループワーク中に参照できる。
- 「倫理的」の反対は「違法」ではない——合法でも倫理に反することがある、という点を押さえる。

### ケーススタディの進め方

**事例①（プラスチックストローの禁止）**

- まずスライドを見せず「プラスチックストロー禁止——あなたはどう思う？」と問う。
- 賛成意見が出た後で「実は困った人がいた」を提示すると対比効果が高まる。
- 「環境チェックはしたが公平チェックが足りなかった」という分析を生徒自身に言わせることが重要。
- 補足：「じゃあどうすれば良かった？」という問いへの答え（例：マイストロー持参の推奨など）をグループで出させると深まる。

**事例②（フードデリバリーの急拡大）**

- 「Uber Eatsや出前館を知ってる？」と問いかけてから始める。
- 「良いサービスでも、拡大したとき新しい問題が生まれる」がこの事例の核心。
- 「持続可能チェック」の重要性が特に伝わる事例なので、スライド13の「アップグレード版」に向けた橋渡しとして使う。

### グループワークの進行

**フェーズ①（アイデア発散）**

- 付箋への書き出しは「1付箋・1アイデア」を徹底する。
- 「問題のブレインストーミング」では評価を禁止する：
  - 「どんな問題でもOK、変なアイデアほど面白い！」
- グループで1つのアイデアを選ぶとき揉める場合：
  - 「多数決ではなく、3チェックでいちばん面白い議論ができそうなものを選ぼう」と誘導。

**フェーズ②（3チェック分析）が最重要**

- チェックA（倫理）で詰まるグループへ：
  - 「このアイデアが実現したとき、喜ぶ人と困る人のリストを書いてみよう」
  - 「弱い立場の人（高齢者・障がい者・子ども・外国籍の方）はどう感じる？」
- チェックB（社会）で詰まるグループへ：
  - 「このアイデアが普及したとき、地域の商店や職業はどう変わる？」
  - 「10年後のこのまちを想像してみよう」
- チェックC（環境）で詰まるグループへ：
  - 「必要な材料は何から作られる？」「使い終わったらどこに行く？」
  - 「同じサービスを1万個・100万個提供したとしたら？」

**フェーズ③（アイデア統合）**

- 3チェックで「問題なし」が全部出たグループ：
  - 「本当に全員にとって問題ない？10年後も同じか？」と問いかける。
  - 「さらに影響範囲を広げる——途上国の人・未来の子どもたちは？」
- 複数の問題が出てどれを解決するか迷うグループ：
  - 「一番影響が大きい問題から解決しよう。最初から全部解決しなくていい」

**フェーズ④（マップ完成）**

- 読めないマップはプレゼンで伝わらない。「発表時に離れても読める大きさで」と声かけ。
- 「10年後の姿」欄は特に具体的に書かせる：
  - 「2036年のこのまちでは〜」「卒業した生徒たちが〜」という書き出しを提案する。

### プレゼンテーション

- 発表フォーマット（スライド15）を板書しておき、全グループが同じ構成で話せるようにする。
- 時間管理：4分経過でやんわり合図、5分経過で終了サインを出す。
- 付箋フィードバックタイムは「沈黙で書く」を徹底する（話しながら書かせない）。
- 発表後のコメントでは、以下の視点で全体フィードバックをする：
  - 「3チェックで最も多く指摘された問題は何だったか」（環境？倫理？社会？）
  - 「グループごとにアップグレードの方向が違う——多様な解決策があることが財産」

### 振り返りシート

- 問3「日常で使う場面」は具体的な場面を提示すると書きやすい：
  - 「文化祭の企画・SNS投稿・買い物の選択・部活のルール作り・学校行事の企画」
- 問4「自律力との接続」は難しく感じる生徒が多い。
  - 「自律力 = 自分で考えて決める力。今日、3チェックを使って自分でアイデアを問い直したよね。あれが自律力だ」と伝えると書きやすくなる。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| グループワークが「雑談」で進まない | 「付箋を書いた枚数を5分後に確認する」と最初に予告すると動き出す |
| 3チェックが「問題なし」ばかりになる | 「100万人が使ったら？」「一番使いにくい人は誰？」と視野を広げる問いを出す |
| アップグレードが「小さな修正」にとどまる | 「根本から変えるとしたら？」「ビジネスモデル自体を変えるとしたら？」と問う |
| プレゼンが「読み上げ」になる | 「マップを指しながら話してみよう」「聴いている人の顔を見よう」と声かけ |
| 環境チェックで「わからない」が続く | 「まず材料を考えよう。その材料はどこから来るか？」と分解させる |
| 振り返り問4（自律力）が書けない | 「自律力 = 自分で考えて決める力。今日体験したどの場面がそれだった？」と問い直す |
| 時間が足りなくてマップが未完成になる | フェーズ③まで完成していればプレゼンは口頭でもOK。マップは参照資料として使う |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **自律力** | 3チェックを使って自分のアイデアを自分で問い直し、改善策を主体的に判断する |
| 創造力 | ケーススタディの事例から学び、サステナブルなアップグレード版アイデアを生み出す |
| 挑戦力 | 「問題あり」を恐れず、改善サイクルを楽しむ姿勢を体験する |
| 協働力 | グループワークで異なる視点を集め、多様な「3チェック」の気づきを統合する |
| 表現力 | マップとプレゼンでアイデアを視覚化・構造化して伝える |
