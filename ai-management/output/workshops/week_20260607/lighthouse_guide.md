## ファシリテーションのポイント

### アイスブレイク（「これって大丈夫？クイズ」）

- クイズは「正解を当てるゲーム」ではなく、「一緒に考えるゲーム」として提示する。
  - 導入例：「正解はないよ。でも、考えてみてほしいことがある。」
- 遊園地クイズでは、以下の気づきを引き出すと効果的：
  - 経営者視点：「無料にしても店は続けられるか？」
  - 公平性視点：「大人や外国人はどう思う？」
  - 持続性視点：「ずっと無料を続けられるか？」
- 「そんな細かいこと気にする？」という反応が出たら：
  - 「細かいことを気にする人が、本当に良いアイデアを作れる！」と返す。
- 補足クイズ（時間が余ったとき）：
  - 「学校でスマートフォン使い放題になったら？」→集中力は？格差は？
  - 「お菓子が給食に出るようになったら？」→体への影響は？農家は？

### スキル紹介（「影響の輪」が核心）

- 「倫理的」という言葉が難しければ「公平かどうか」「誰かが傷ついてないか」に言い換える。
- 「持続可能」は「10年後もそのアイデア使える？」という問いに翻訳すると理解しやすい。
- スライド5の図を指でなぞりながら：
  - 「みんなが最初に考えるのはこの輪（直接助かる人）だけ。でも実は……」
  - 「影響は広がっていく。それを最初から考えられる人が、本当の起業家だ」
- 具体例を1つ持っておくと強い：
  - 「コンビニのプラスチックスプーンをやめた——最初は不便だと思った人もいたけど、地球への影響を考えた結果」

### アイデア・アップグレードシート記入のポイント

**ステップ1（アイデアの種）**
- 「大きな社会問題じゃなくていい」「今朝から学校に来るまでに気になったことでOK」と伝える。
- 書けない子へのサポート例：
  - 「給食で困ったことある？」「通学路で不便だと思ったこと、ある？」
  - 「最近誰かに『これ、なんとかならないかな』と思ったことは？」
- 「アイデアの名前」は後でつける（先に内容を書いてから命名させる）。

**ステップ2（3チェック）が最重要**
- 「問題あり」にチェックが入った子を必ず褒める：
  - 「その気づき、すごい！それができる人が本物の設計者。」
- 「わからない」にチェックが入った子にも：
  - 「わからないと思えることが大事。だから調べたり、聞いたりすることが必要になる。」

**チェック① 公平チェック の引き出し方**
- 「このアイデアで、お年寄りはどう感じる？」
- 「障がいのある人は使えるかな？」
- 「お金が少ない家族は使える？」

**チェック② 地球チェック の引き出し方**
- 「このアイデアを実現するとき、何かを消費する？（電気・紙・プラスチック）」
- 「たくさん作ると、ゴミはどうなる？」
- 「このアイデアが100万人に広まったとき、地球はどうなる？」

**チェック③ 未来チェック の引き出し方**
- 「このアイデア、10年後も使われてると思う？」
- 「技術が変わっても、このアイデアの考え方は残る？」
- 「このアイデアで、誰かが依存しすぎたり、困ったことにならない？」

**ステップ3（アップグレード）**
- 「問題なし」が全部の子が実は一番難しいケース：
  - 「じゃあ、あと1つ輪を広げてみよう。まだ幸せになれていない人、いないかな？」
  - 「世界中でこれが使われたら？」と視野を広げる問いを出す。
- アップグレードしても新しい問題が生まれることがある：
  - 「完璧にならなくていい。改善のサイクルを回せることが大事、と伝える。

### 巡回中のポイント

- 「公平チェック」でつまずいているチームへ：
  - 「このアイデアを使う人の中で、一番使いにくそうな人を1人想像してみよう」
- 「地球チェック」で「わからない」が続くチームへ：
  - 「材料は？電気は？ゴミは？この3つを確認してみよう」
- アップグレードが「もっと機能を増やす」だけになっているチームへ：
  - 「機能を増やさず、対象を広げる方法はある？」

### 全体発表

- 板書の「3チェック表」に各チームの発見を書き込んでいくと、全体で「倫理的・持続可能な課題マップ」が完成する。
- 同じ問題（例：「環境への影響」）が複数チームから出たとき：
  - 「みんながここで同じ問題に気づいた——それは、現代の重要な課題のひとつ！」
- 異なる問題が出たとき：
  - 「視点が違う——それが多様なチームの強さ！」

### 振り返りシート

- 「書けない」子には「今日の3チェックで一番驚いたものは？★だけ先につけて」と誘導する。
- 問2「アイデアがどう変わったか」は全体でシェアすると共感が生まれる：
  - 「アイデアが変わった人？手を挙げて。どんなふうに変わったか1文で教えて」
- 「ひとこと感想」欄は「次に何かアイデアを出すとき、一番最初に思い出すことを書こう」と伝えると書きやすい。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| 「3チェックって何が問題なの？」と難しがる子がいる | 「たった3つの質問に答えるだけでいい。まず①から一緒にやってみよう」 |
| アイデアが思い浮かばない | 「今日の朝ごはんで不便なことは？」「通学路で嫌だったこと、ある？」と具体化 |
| 「問題なし」ばかりで考えが止まる | 「100万人が使ったとしたら？」「お年寄りが使うとしたら？」と別の視点を提供 |
| アップグレードが「ちょっと変えた」だけになる | 「何が根本的に変わった？」「誰が新しく幸せになった？」と深掘りする |
| 発表を嫌がる子がいる | 「シートを見せるだけでもOK。読み上げなくていい」と声かけ |
| 「倫理的」「持続可能」の言葉が難しい | 「公平か」「ずっと続けられるか」に随時言い換えて使う |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **自律力** | 自分のアイデアを自分で問い直し、主体的に改善策を考え抜く |
| 創造力 | 3チェックの発見からアップグレード版のアイデアを生み出す |
| 挑戦力 | 「問題あり」という発見を恐れず、改善を楽しんでみる |
| 協働力 | チームのフィードバックで気づけなかった視点を得る |
| 表現力 | アップグレードしたアイデアを「誰が・どう幸せになるか」で言語化する |
