# スライド2: アイスブレイク（5〜10分）

## 「これって大丈夫？」クイズ！

**ファシリが問題を出す。チームで「大丈夫 or 問題あり」を考えよう！**

---

**問題① ある遊園地が「子どもは全員無料！」を始めた。**

> 素晴らしい！…でも、本当に大丈夫？

**チームで考えよう（2分）：**
- 誰かが困る人はいないかな？
- 何か問題は起きないかな？

---

# スライド3: アイスブレイク 解説

## 遊園地クイズの答え

**良いところ：**
- 子どもが遊べる！嬉しい！

**でも、問題あり：**
- 遊園地のスタッフのお給料はどうなる？
- お客さんが増えすぎて遊具が壊れないか？
- 無料分のお金は誰が払うの？

**気づいた？**

> ひとつのアイデアには、**見えていない影響**がある。
> 「誰かが喜ぶ」と「誰かが困る」は、セットで考えることが大事。

# スライド4: スキル紹介（5分）

## 今日のスキル：「倫理的・持続可能な思考」

**EntreComp（起業家に必要な力）より**

> 自分のアイデアが「誰かを傷つけていないか」
> 「地球・社会にとって続けられるか」を考える力

**2つのキーワード：**

| キーワード | 意味 |
|----------|------|
| **倫理的** | みんなに公平で、誰も不当に傷つけない |
| **持続可能** | 今だけじゃなく、未来もずっと続けられる |

🌊 賢い航海士は、自分の船を進めるだけでなく、
　　海の環境と仲間の船のことも考える。

# スライド5: なぜ大事なの？（5分）

## アイデアには「影響の輪」がある

```
         自分のアイデア
              ↓
     ┌────────────────┐
     │ 直接助かる人   │← これだけ考えがち
     └────────────────┘
              ↓
     ┌────────────────┐
     │ 間接的に影響   │← 実はここが大事
     │ を受ける人     │
     └────────────────┘
              ↓
     ┌────────────────┐
     │ 環境・地球・   │← 未来のことも！
     │ 未来の人たち   │
     └────────────────┘
```

**良いアイデアとは：3つの輪すべてが笑顔になること**

# スライド6: メインワーク説明（2分）

## 「アイデア・アップグレードシート」に挑戦しよう！

**今日のワーク（4ステップ）**

1. 「誰かの問題を解決するアイデア」を1つ出す
2. **3チェック**で「隠れた問題」を探す
3. 問題が見つかったら「アップグレード版」を考える
4. チームで発表して、フィードバックをもらう

**準備するもの**
- アイデア・アップグレードシート（配布）
- えんぴつ or マーカー

⏱ **ステップ1〜3 合わせて30分。さあ、スタート！**

---

# スライド7: ステップ1「アイデアを出そう」（5分）

## シートの上段：アイデアの種

**シートに書く：**

```
【アイデアの種】

① 解決したい問題・不便なこと：
  →

② 誰のため？（ターゲット）：
  → （例：小学生・お年寄り・忙しいお父さん）

③ どんな解決策？（アイデア）：
  →

④ このアイデアの名前：
  →
```

✏️ **5分！なんでもOK。学校のこと・家のこと・街のことから選ぼう。**

---

# スライド8: ステップ2「3チェックをしよう」（15分）

## シートの中段：アイデアの健康診断

**3つのチェックに答えよう：**

```
【チェック① 公平チェック】
このアイデアで、誰かが損したり、
不公平になる人はいないかな？

答え → □ 問題なし  □ 問題あり

問題がある場合、それは誰？ どんな問題？
→


【チェック② 地球チェック】
このアイデアは、環境や地球に
悪い影響はないかな？
（ゴミが増える、エネルギーを大量に使うなど）

答え → □ 問題なし  □ 問題あり  □ わからない

→


【チェック③ 未来チェック】
このアイデアは、10年後も
続けられる？ 使い続けられる？

答え → □ 続けられる  □ 問題あるかも  □ わからない

→
```

💡 **「問題あり」が出ても大丈夫！それが次のステップへの扉。**

---

# スライド9: ステップ3「アップグレードしよう」（10分）

## シートの下段：アイデアをバージョンアップ！

**チェックで「問題あり」が出た場合：**

```
【アップグレード版アイデア】

どの問題を解決することにした？
→ □公平  □地球  □未来

どうアップグレードした？
→


アップグレード後のアイデア名（変えてもOK）：
→


アップグレードで「新しく幸せになる人」は誰？
→
```

⭐ **チェックで全部「問題なし」だった人は：**
「さらにもっと多くの人を幸せにする方法」を考えよう！
