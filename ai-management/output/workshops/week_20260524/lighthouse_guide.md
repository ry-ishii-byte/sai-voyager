## テーマ: 機会と価値の創造
**EntreComp領域**: 機会の発見
**SAILの力**: 創造力

**所要時間**: 60〜90分
**推奨人数**: 12〜30名（3〜5名/チーム）

## ファシリテーションのポイント

### アイスブレイク

- 「不便だな」を言いにくい雰囲気を作らないよう、最初にファシリ自身が1つ披露する。
  - 例：「私は傘を店に忘れてくることが多くて、傘立てに名前を書くシールがあればいいのに、と思ってます！」
- 「クレームじゃなくてOK」「ちょっとしたことでいい」と伝えると参加しやすくなる。
- 「そう、私も！」という反応が出たら「その共感、すごく大事！共感 = 同じ問題を感じてる人がいる ということ」とつなげる。

### スキル紹介（価値の説明が核心）

- 「価値」という言葉が難しければ「誰かのうれしい！を作ること」に言い換える。
- スライド5の流れ図を指でなぞりながら「問題を見つける → 解決する → 誰かがハッピーになる。この流れが"価値を創る"ということ」と伝える。
- 「お金持ちになるために起業するんじゃなくて、誰かのために問題を解決するのが起業家の本質」というメッセージを加えると深まる。

### チェンジメーカーカード記入のポイント

**問題発見カード（ステップ1）**
- 「大きな社会問題じゃなくていい」「今朝起きてから感じた不便でOK」と伝える。
- 書けない子へのサポート例：
  - 「学校のロッカーが使いにくいと思ったことある？」
  - 「給食で『これ不便だな』と思うことは？」
- 「誰が困っている？」は「自分だけじゃなくて、他の人も困ってるかな？」と考えさせると視野が広がる。

**価値の創造カード（ステップ2）が核心**

- 「完璧なアイデアじゃなくていい。テクノロジーがなくてもいい。"こんなのがあれば"で書いてOK」と繰り返す。
- アイデアに詰まっている子へ：「もし魔法が使えたら、どう変える？」と問いかける。
- 「誰がハッピーになる？」は具体的な人物像で答えさせると深まる。例：「80歳のおじいちゃん」「足の不自由な友達」など。
- 「アイデアに名前をつけよう」の指示が最重要。命名することで「自分のアイデア」という当事者意識が生まれる。

### 巡回中のポイント

- 「誰が困っているか」が「みんな」になっているチームへ：「もう少し絞ると？例えば学校に来てる子？お年寄り？」と具体化させる。
- アイデアが「アプリを作る」だけになっているチームへ：「アプリ以外の方法もあるかな？」と広げる。
- カードを書き終えて話が弾んでいるチームへ：「その話、カードに書き足してみよう！」と声かけ。

### 全体発表

- 「アイデアの名前」を板書・ホワイトボードに書き出すと、全体の「価値の多様性」が見える化されて盛り上がる。
- 似た問題を違う方法で解決しているアイデアが出たら「同じ問題でも解決策は無限にある！」とつなげる。
- 「使ってみたい！」という反応が出たら「それが"価値"が生まれた瞬間！」とリアルタイムで強調する。

### 振り返りシート

- 「書けない」子には「今日書いたカードを見ながら、★だけ先につけてみて」と誘導する。
- 問2「どちらが難しかったか」は全体でシェアすると共感が生まれる。「問題発見の方が難しかった人？」→「アイデアの方が難しかった人？」と手を挙げさせると面白い。
- 「ひとこと感想」欄は「明日の自分へのメモ」として書くよう伝えると書きやすい。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| 「不便なことがない」という子がいる | 「今朝から学校に来るまでに、何か面倒だったことは？」と具体的に聞く |
| アイデアが「スマホで解決」ばかりになる | 「スマホ以外だったら？道具や仕組みで解決するとしたら？」と広げる |
| 「誰が困っている」が抽象的（「世界の人」など） | 「具体的に、誰か1人を思い浮かべよう。名前をつけてもいいよ」 |
| 発表を嫌がる子がいる | 「チームの代表として発表してもOK。自分のカードを見せるだけでもOK」 |
| チームで話し合いが止まる | 「一番面白いアイデアに★をつけよう」と投票形式を使う |
| 振り返りシートが白紙のまま終わる | ★の数だけ先につけてもらい「なんで★3にした？」と引き出す |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **創造力** | 問題を発見し、新しい解決アイデアを生み出す |
| 自律力 | 自分で問題を見つけ、自分でアイデアを考え抜く |
| 挑戦力 | 「完璧じゃなくていい」と一歩踏み出して書いてみる |
| 協働力 | チームでアイデアをシェアして互いを刺激し合う |
| 表現力 | チェンジメーカーカードと発表でアイデアを言語化する |
