# 外部版 指導者ガイド
## テーマ: 倫理的・持続可能な思考
### EntreComp スキル#4 ｜ 機会の発見 / 自律力
### 対象: 小中学生（学校・自治体向け） ｜ 時間: 120〜180分

---

## 概要・ねらい

「倫理的・持続可能な思考」（EntreComp スキル#4）をテーマに、社会課題と向き合うアントレプレナーシップ教育を実施します。学校・自治体向けの本プログラムは、より深い対話と探究を重視した設計です。ケーススタディと「未来のルールを作る」グループワークを通じて、「社会全体を見渡した責任ある選択」を体験的に学びます。

**到達目標**:
- 倫理的・環境的・社会的視点を組み合わせて物事を評価できる
- 自分の意見を根拠とともに表現できる
- 多様な意見を統合して提言できる（協働力）

**SAILの5つの力との接続**:
- **自律力（主）**: 自分の価値観基準で判断する
- **表現力（主）**: 提言書・プレゼンで論理的に伝える
- **協働力（副）**: グループで合意形成する
- **創造力（副）**: 制約の中で実現可能なルールを発案する

## セクション別進行メモ

### [0:00〜0:10] アイスブレイク

**目的**: 「価値判断の難しさ」を体感し、本日のテーマへの問題意識を高める

**進行**:
1. 「立ってください」と全員を起立させる
2. 「直感で答えてください、考えすぎずに」と前置き
3. 質問を3問読み上げる
   - 「安くて地球に優しくない商品 → 右、高くて地球に優しい商品 → 左に動いてください」
4. 各問の後、数人に理由を聞く（一言）
5. 「判断が難しかったのはなぜか？」を問いかけ2〜3人の声を拾う

**ファシリポイント**:
- 価値観の多様性・情報不足・トレードオフの3点が出てくるよう誘導する
- 「正解・不正解はない。でも考えることをやめないことが大事」と締める

---

### [0:10〜0:30] 導入講義

**目的**: 倫理的・持続可能な思考の概念と必要性を理解する

**スライド4〜7の講義ポイント**:

**EntreCompの説明（スライド4）**:
- EUが2016年に開発したフレームワーク
- 起業家精神 = ビジネスだけでなく「社会を変える力」を含む
- SAILの5つの力との対応を図示しながら説明

**現代社会の3問題（スライド5）**:
- トレードオフ: 「安い服はなぜ安い？」と問いかけ、サプライチェーンの話へ
- 時間軸のズレ: 「今のプラスチック使用が100年後の海を汚す」具体例
- 見えにくい影響: 「自分のスマホはどこで誰が作った？」

**3つの持続可能性（スライド6）**:
- 図を指しながら「3つがバランスしているのが理想」
- 「どれかに偏ると破綻する」例を挙げる
  - 環境重視のみ: 経済が回らず普及しない
  - 経済重視のみ: 環境・社会が壊れる

**世界事例（スライド7）**:
- パタゴニアは「利益を減らすが長期信頼で成長」→ 「これはビジネスとして成立している」
- グラミン銀行は「問題ではなくシステムを変えた」→ 倫理的思考の視点転換

**ファシリポイント**:
- 教壇から離れ、聴衆に近い場所で話す
- 質問を随時投げかけ、双方向にする
- 専門用語（サプライチェーン等）は毎回平易な言葉で言い換える

---

### [0:30〜1:00] ケーススタディ

**目的**: 現実の企業行動を倫理的視点で評価し、判断の複雑さを体験する

**進行**:
1. ケーススタディシートを配布（5分）
2. スライド8を投影し、ケースAまたはBを読み上げる
3. 個人評価（5分）
4. ペアシェア（5分）: 評価を共有し相違点を確認
5. グループ討論（10分）: チームとしての評価と「最も意見が割れた点」を選ぶ
6. 全体共有（10分）: 各グループから「意見が割れた点」を発表 + ファシリがまとめ

**ファシリポイント**:
- ケースAとケースBの両方をやる場合はグループで分ける（半分/半分）
- 「どっちの企業が悪いか」という善悪二元論にならないよう注意
- 「どうすればより良くなるか」という改善の視点を持ち込む
- 全体共有では板書しながら横断的なパターンを整理する
  （例: 「みんな労働問題を指摘した → なぜ労働問題が多いと思う？」）

**中学生向け補足**:
- CSR（企業の社会的責任）・SDGs・ESG投資について補足説明できると深まる
- 「消費者としての私たちの選択も影響する」と個人レベルへの接続を忘れずに

---

### [1:00〜2:00] グループワーク: 未来のルールを作ろう

**目的**: 倫理的・持続可能な視点から、具体的な提言を生み出す

**グルーピング**:
- 4〜5人1チームが理想
- 業界が重複してもよいが、多様なチームになるよう促す

**Step 1 業界選択と問題特定（10分）**

進行:
- スライド11の業界候補を提示
- チームで業界を1つ選ぶ（多数決OK）
- 選んだ業界の「倫理的・環境的問題」を3つ特定し、ポスター上段に書く

ファシリポイント:
- 「身近な例で考えて。コンビニ、学校、スポーツでも何でもいい」
- 問題の深さより数（3つ）を優先させ、後で絞る
- 詰まっているチームにはヒントカードを手渡す

**Step 2 ルール案の考案（15分）**

進行:
- 問題の中から「最も解決したい問題」を1つ選ぶ
- その問題に対する「ルール案」を3つ以上考える
- 付箋に1つずつ書き出す（アイデア発散）

ファシリポイント:
- 「法律みたいに具体的に！『○○は禁止』『○○%以上は〜』等」
- 実現可能性は後で考えるよう伝え、まず発散させる
- 「誰がこのルールを守るか」の主語も意識させる

**Step 3 3持続性チェック（15分）**

進行:
- ルール案それぞれを3持続性チェックカードで評価
- ○/△/×をつけ、理由を一言書く
- 全部○のルールを「提言ルール」として選ぶ（全部○がない場合は△が多いものを改良）

ファシリポイント:
- 「経済×のルールは企業が従わないから意味がない → どう改良する？」
- 「社会×のルールは誰かを傷つける → それは持続可能？」
- チーム内でルールを組み合わせることを促す（2つのルールの良いとこ取り）

**Step 4 提言書作成（20分）**

進行:
- スライド13のフォーマットに従ってA3ポスターを作成
- タイトル → 問題の背景 → 3つのルール → 期待効果 → チーム名

ファシリポイント:
- 10分で下書き → 5分でデザイン → 5分で見直し、のリズムを伝える
- 「審査員に伝わるか、発表前提で作ろう」と意識を高める
- 時間が足りないチームは「ルール2つ + 背景1つ」に絞る

---

### [2:00〜2:30] プレゼンテーション

**目的**: 思考プロセスを言語化し、他者に伝える（表現力）

**進行**:
1. 発表順をランダムに決める（くじ引き等）
2. 各チーム4〜5分発表 + 2〜3分質疑
3. 聴衆チームは発表中「良い点」「質問したい点」を付箋にメモ
4. 全チーム終了後、ファシリがテーマ横断の学びをまとめ

**評価の観点（発表後にフィードバック）**:
- 問題の把握は具体的か
- ルールは実現可能か
- 3持続性の視点が反映されているか
- 発表は論理的か

**ファシリポイント**:
- 発表者の勇気を称える（「よく考えてくれました」等）
- 批判的な質問が出た場合は「それは良い問い」と評価してから発表者に返す
- 同じ業界のチームがいれば「どう違うアプローチをとったか」を比較する

---

### [2:30〜2:50] 振り返り・自己評価

**目的**: 体験を言語化し、内面化する

**進行**:
1. 振り返りシート記入（5〜7分）
2. グループ内でシェア: 「今日変わった考えを1つ」（5〜7分）
3. 全体へ3〜4名が発表（自由意志）
4. 自己評価シート記入（3〜5分）

**ファシリポイント**:
- 「良いこと言わなくていい。正直に書こう」と促す
- 「考えが変わった」「難しかった」等、正直な振り返りを称える
- 「今日使ったSAILの力は何？」を必ず問いかける（自律力・協働力等）

---

### [2:50〜3:00] まとめ

**進行**:
1. スライド17〜18を使ってSAILメッセージを伝える
2. 「今日持ち帰る3つの問い」を板書しながら強調
3. 学校担当者・自治体担当者への挨拶・案内
4. アンケート配布（実施している場合）

## トラブルシューティング

| 状況 | 対応 |
|---|---|
| 講義中に質問が止まらない | 「後でまとめてお答えします」と付箋に書いてもらう |
| ケーススタディが難しすぎる場合 | 「自分の好きなお店やサービスで同じ問題はある？」と身近化 |
| グループワーク中に意見が対立 | 「どちらの意見も正しい。どう組み合わせる？」と調停 |
| 時間が大幅に遅れた場合 | Step 3の3持続性チェックを口頭確認に変更、Step 4を簡略化 |
| 発表が少数チームに偏る | 「次の発表は○グループでお願いします」と指名 |
| 自己評価シートを書けない生徒がいる | 「1〜2項目だけでOK、今日何か感じたことを書いてみて」 |
