# スライド3: 今日のテーマ

## アントレプレナーシップと「資源」

アントレプレナー（起業家）に必要な力はたくさんある。
その中の一つが「**資源を見つけ、活かす力**」。

**なぜ重要か**:
- 新しいことを始めるとき、最初から全部揃っていることはない
- 「足りないもの」を並べれば行動できない→「あるもの」を活かせば今日から動ける
- チームで力を合わせると、個人の限界を超えられる

これが「資源の動員 / 協働力」の本質。

# スライド5: 3種類の「資源」

## モノ・ヒト・コトで整理する

### モノの資源（Physical Resources）
- 道具・機材・スペース・インフラ
- 例：学校の空き部屋、スマートフォン、地域の公民館

### ヒトの資源（Human Resources）
- 人の得意なこと・つながり・ネットワーク
- 例：絵が得意な仲間、地域の専門家、SNSのフォロワー

### コトの資源（Cognitive/Social Resources）
- 知識・経験・信頼・共感
- 例：自分たちが経験した問題への共感、調査データ、地域の歴史

「見えにくい資源ほど、実は強力なことがある」

# スライド6: ケーススタディ（20〜30分）

## 「制約の中で動いた起業家たち」

### ケースA: 地方の高校生が始めた地域活性化プロジェクト

**状況**: 人口減少が続く町。廃校になった小学校の建物がある。
**持っていた資源**: 廃校の建物 ＋ 地域のおじいちゃん・おばあちゃんの知恵 ＋ SNSを使える高校生たち ＋ 地元の木材
**やったこと**: 高校生がファシリとなり、お年寄りの知恵を「ものづくりワークショップ」として発信。廃校をイベント会場に。
**結果**: 都市部から参加者が来るようになり、地域外収入が生まれた

---

# スライド7: ケーススタディ（続き）

### ケースB: 中学生が学校の「給食ロス問題」を解決

**状況**: 給食の食べ残しが毎日大量に出ている学校
**持っていた資源**: 生徒の「もったいない」という気持ち ＋ 調理員さんのレシピ知識 ＋ 学校の放送設備 ＋ アンケート経験
**やったこと**: アンケートで「なぜ残す？」を調査 → 調理員と協力して人気メニューの頻度を上げる提案 → 放送で「おいしいポイント」紹介
**結果**: 食べ残し率が3ヶ月で約30%減少

**ディスカッション（10分）**: 2つのケースで「最も効いた資源」はどれだったか？なぜ？

# スライド8: ケーススタディ振り返り

## 気づきの整理

**問い**（班で3分、全体シェア7分）:
1. 2つのケースで、最初に「なかったもの」は何だった？
2. それを補ったのはどんな「あるもの（資源）」だった？
3. もし自分たちのチームなら、どんな資源を使ったか？

「資源は探す意図があって初めて見える」

---

# スライド9: グループワーク説明（40〜60分）

## 「地域課題に挑む：資源マップから解決策へ」

**テーマ**: 自分たちの地域・学校・日常にある「小さな課題」を1つ選び、手元の資源だけで解決策を設計する

**チーム**: 4〜5人

### フロー:
- Phase 1（15分）: 課題の選定 ＋ 課題の深掘り（なぜ起きているか？誰が困っているか？）
- Phase 2（20分）: 資源マップ作成（モノ・ヒト・コトを付箋で整理）
- Phase 3（15分）: 資源を組み合わせた解決策を3案考え、1つに絞って具体化
- Phase 4（10分）: 発表スライド（A3）またはプレゼンボード作成

# スライド11: Phase 2 資源マップ作りのガイド

## 「見えにくい資源」を掘り起こすための質問

**モノについて**:
- 学校・地域にある使われていないスペースや設備は？
- デジタルツール（SNS・アプリ・Webサイト）は使えるか？

**ヒトについて**:
- チームメンバーそれぞれの「強み・得意なこと」は何？
- 話を聞いてくれそうな大人（先生・保護者・地域の人）は誰？
- 問題に詳しい専門家や当事者はいるか？

**コトについて**:
- チームで共通して「怒り・もったいない・面白い」と感じていることは？
- 過去に似た問題に取り組んだ事例を知っているか？
- 自分たちが発信できる「共感ストーリー」はあるか？

---

# スライド12: Phase 3 アイデアの設計

## 「資源 × 課題」で解決策をデザインする

**思考フレーム**:
```
もし ______（資源）があれば、
______（課題）に対して
______（アクション）ができる。
そうすれば ______（誰が）が ______（どう変わる）。
```

**絞り込みの基準**:
- 今ある資源だけで始められるか？（外部資源への依存度が低いか）
- 3ヶ月以内に「最初の一歩」が踏み出せるか？
- チームメンバーの誰かが「これをやりたい！」と思えるか？

---

# スライド13: プレゼンテーション（20〜30分）

## 各チーム発表

**発表構成（1チーム4〜5分）**:
1. **課題**: 何が問題か、誰が困っているか（30秒）
2. **資源マップ**: 使った主な資源3つ（30秒）
3. **解決策**: 何をするか、なぜこの資源を選んだか（2分）
4. **最初の一歩**: 今週から始められることは何か（30秒）

**聞いている人へ**: 「使えそうなアイデア」「もっと活かせる資源」があればコメントしよう
