## ファシリテーションのポイント

### アイスブレイク
- 付箋を書くとき「正解はない」と必ず伝える。「ゲームが好き」「お喋りが得意」なんでもOK。
- 苦手カードを書くのを嫌がる子には「得意カードだけでOK」と伝える。
- 得意付箋をチームでテーブルに並べるとき「似てるやつある！」と盛り上がるよう促す。

### スキル紹介
- 「みんなが知ってるチームの成功例」を問いかける。スポーツ・アニメ・学校行事など子どもが答えやすいものを想定しておく。
- 答えが出ない場合の例: 「ラグビーとか、サッカーとか、ちがう役割の人がそろうよね」

### PART1（1人チャレンジ）
- 「難しくて当然！」を最初に伝えておく。「3分で完璧にしなくていい、何か書ければOK」。
- 焦らせない。静かな3分間を保証する。

### PART2（チームワーク）
- チームを巡回し、1人が黙っているチームには「○○さんのアイデアも聞いてみよう」と促す。
- 1人が独走しているチームには「他の人はどう思う？」と向ける。
- 「意見が分かれていい、それがチームの強さ」とスライド9を参照しながら声かけ。

### 発表タイム
- 全チームが終わったら「他のチームで好きだったポイントを1つ言える人？」と拾う。
- ファシリが率先してリアクション（「それ面白い！」）を見せてモデルになる。

### 振り返りシート
- 書けていない子には「今日、自分が一番チームの役に立った瞬間はいつ？」と聞いて埋めてもらう。
- 「★いくつつけた？」と声に出して共有させると場が温まる。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| チームで誰も発言しない | 「じゃんけんで勝った人から言ってみよう」でハードルを下げる |
| 1人が独占してしまう | 「他の人のアイデア、どれが好きだった？」と全員へ振る |
| アイデアが出ない | 「もし自分がこの商品のCMを見たらどう感じる？」と聞く |
| 仲良しチームが内輪で盛り上がりすぎ | 発表時に「初めて見た人にも伝わるか？」を問いかける |
| 振り返りシートが書けない | ★の数だけ先に書いてもらい、「なんで？」と引き出す |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **協働力** | 1人 vs チームで成果の差を体感する。役割分担・意見調整を実践 |
| **表現力** | CMという形式でアイデアを「伝わる形」にする |
| 創造力 | 架空商品のCMというフレームで創造性を刺激 |
| 自律力 | 振り返りシートで自分の貢献・成長を自己評価 |
| 挑戦力 | 「1人では難しい」状況に飛び込む体験 |
