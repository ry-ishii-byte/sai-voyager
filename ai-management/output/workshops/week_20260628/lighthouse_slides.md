# スライド5: 資源って何？

## 資源 ＝ チカラになるもの全部！

| 種類 | 例 |
|------|-----|
| 💰 お金 | おこづかい、賞金 |
| ⏰ 時間 | 放課後、休み時間 |
| 👥 人・つながり | 友だち、家族、先生 |
| 📚 知識・スキル | 得意なこと、趣味 |
| 🏫 場所・モノ | 教室、公園、図書館 |

**「お金がないとできない」は思い込みかも？**

# スライド7: メインワーク①「資源マップ」

## あなたの資源を全部出そう！

シートに書き出そう **（10分）**：

1. **自分のスキル・得意なこと** → 3つ
2. **使える場所** → 2つ
3. **頼れる人** → 3人の名前
4. **持っているモノ** → 3つ

> **ヒント**: 「ふつうのこと」でOK！料理が好き、ゲームが得意も立派な資源！

# スライド8: メインワーク②「アイデア出し」

## 資源を組み合わせてみよう！

グループ（4〜5人）で話し合おう **（20〜25分）**

**テーマ：「この資源を組み合わせると、どんな面白いことができる？」**

---

## ルール 3か条
1. どんなアイデアでも **OK！**
2. 「それは無理」は **禁止！**
3. 量を出す → あとで選ぶ

> 付箋に1つずつ書いて貼っていこう！

# スライド10: まとめ

## 今日の3つの学び

**① 資源はお金だけじゃない**
時間・スキル・人つながり・場所…全部資源

**② 「あるもの」でスタートできる**
ないものを嘆かず、今あるものを使い切ろう

**③ 組み合わせると新しい価値が生まれる**
一人の資源 ＜ みんなの資源の組み合わせ

**あなたにも、すでにたくさんの「資源」がある！**
