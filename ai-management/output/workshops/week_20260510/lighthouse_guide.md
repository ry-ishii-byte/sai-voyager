# LIGHTHOUSE版 指導者ガイド
## テーマ: 経験からの学び
**EntreComp領域**: 不確実性への対処
**SAILの力**: 自律力
**対象**: 小中学生（小4〜中3想定）
**所要時間**: 60〜90分
**推奨人数**: 12〜30名（3〜5名/チーム）
**生成日**: 2026/05/10

## ファシリテーションのポイント

### アイスブレイク

- 「失敗の話をしてもらう前に」必ず「笑わない・うなずく」を全体に宣言する。心理的安全性がこのワークの命。
- ファシリ自身が最初に短い「失敗話」を披露すると、参加者が話しやすくなる。
  - 例：「私も昔、人前でプレゼン中に資料を逆さに持ってた！あれで学んだのは確認の大切さ（笑）」
- 「失敗がない」という子には「最近困ったことでもOK」と返す。

### スキル紹介（ループ図の説明）

- スライド5の4ステップループは、指やペンで「矢印をなぞる」動作をしながら説明すると子どもに伝わりやすい。
- 「ループ」という言葉がわかりにくい場合は「グルグル回る力の法則」と言い換えてもOK。
- 「経験は『する』だけでは育たない。『ふり返る』ことで学びになる」というメッセージを2回繰り返す。

### 失敗カード記入のポイント

- 「大きな失敗でなくていい」「昨日の失敗でもOK」と伝えると書きやすくなる。
- 書けない子へのサポート例：
  - 「最近、思ったよりうまくいかなかったこと、何かある？」
  - 「学校でも家でも、なんでもOKだよ」
- 書き終わった子が他の子の邪魔をしないよう、「書けた人は静かに待とう」と伝えておく。

### 発見カード記入のポイント（核心）

**このワークの最大の目的は「失敗を見方を変えて捉え直す」体験。**

- 「発見の名前をつけよう」の指示が最重要。命名することで「自分だけの学び」になる。
  - 例：「準備は早めが勝ち」「人に聞くのを恥ずかしがらない」「やってみないとわからない」
- 「なぜそうなったか」に詰まっている子へ：「もし同じ状況が明日来たら、最初に何を変える？」と問いかける。
- 「次はこうしよう」に「とにかく頑張る」と書いてしまう子へ：「具体的に何をする？1つ教えて」と絞り込ませる。

### 巡回中のポイント

- 「発見カード」がポジティブすぎてふわっとしているチームへ：「それ、なぜそう発見したの？」と掘り下げる。
- チーム内で話が弾んでいるが、書く手が止まっているチームへ：「いいこと言ってる！それ書いてみよう！」とカードに向かわせる。
- 一人が話し続けているチームへ：「次の人の発見も聴きたいな」と自然に切り替える。

### 全体発表

- 「発見の名前」を黒板やホワイトボードに書き出すと、全体の学びが「見える化」されて盛り上がる。
- 似た発見が複数出たら「みんな同じことを発見したんだ！」とつなげる。
- 珍しい発見が出たら「それ、面白い視点！」と必ず拾う。

### 振り返りシート

- 「書けない」子には「今日の失敗カードを見ながら、★だけ先につけてみて」と誘導する。
- 「ひとこと感想」欄は「未来の自分へのメッセージ」として書くよう伝えると書きやすい。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| 「失敗したことない」という子がいる | 「困ったこと・思い通りにいかなかったこと」に言い換える |
| 失敗カードに書いた内容が「恥ずかしい」と共有を拒む | 「発見だけ発表してもOK。失敗の内容は見せなくていいよ」と伝える |
| 発見カードに「次は頑張る」しか書けない | 「頑張る = 具体的に何をする？」と問い返す |
| チーム内で話が盛り上がりすぎて時間オーバー | 「もう1分で切るね」と予告してから止める。内容は全体発表に持ち越す |
| 振り返りシートが白紙のまま終わる | ★の数だけ先につけてもらい「なんで★3にした？」と引き出す |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **自律力** | 自分の経験を自分でふり返り、学びを発見する |
| 挑戦力 | 「失敗」をオープンにして、次につなげる勇気 |
| 創造力 | 「発見の名前」をつけることで学びを言語化・新解釈する |
| 協働力 | チームで発見を共有し、互いの学びを豊かにする |
| 表現力 | 「発見カード」の発表で学びを言葉にして伝える |
