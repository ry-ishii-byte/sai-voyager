# スライド3: アイスブレイク「強みオークション」（10分）

## 自分の強みはいくら？

**やり方**:
- 架空の「強みポイント」100ptを持っている
- 「強みリスト」から「自分にある強み」を選び、持ちポイントを使って競り落とす
- 最後に「なぜその強みを選んだか」を1人30秒で話す

**強みリスト（例）**:
> 粘り強さ / 共感力 / アイデア力 / 実行力 / 観察力 / 表現力 /
> チームワーク / リーダーシップ / 細かさ / 楽観性

---

# スライド4: 自己認識とは

## 「自分のトリセツ（取扱説明書）」を持つこと

自己認識の3つの次元:

| 次元 | 問い | なぜ重要か |
|---|---|---|
| **強み** | 自分が得意なことは何？ | 資源の最大活用 |
| **価値観** | 自分が大切にしていることは？ | 判断・行動の軸 |
| **経験** | 乗り越えた・成長した出来事は？ | 根拠ある自信の源泉 |

「自分をよく知る人は、どんな状況でも舵を自分で握れる」

---

# スライド5: 自己効力感とは

## 「私にはできる」という確信の力

**自己効力感が高い人の特徴**:
- 難しい課題に積極的に取り組む
- 困難に直面しても粘り強く努力する
- 失敗を「次への学習」として捉える

**低い人との違い**: 能力の差ではなく、「信念」の差

> 「特定の課題を達成できるという自分自身への信念」

## 自己効力感の4つの情報源

# スライド7: ケーススタディ（20〜30分）

## 強みが世界を変えた3つのストーリー

**Case A: グレタ・トゥーンベリ（環境活動家）**
- 強み: 「おかしいと感じたら、行動せずにいられない」
- 自己認識: ASD（自閉症スペクトラム）を「スーパーパワー」と呼んだ
- 結果: 世界中の若者を動かした気候変動運動

**Case B: 吉藤オリィ（OriHime開発者）**
- 強み: 「孤独を知っている。ロボットが好き。人を助けたい」
- 自己認識: 不登校・入院体験が「使命感」になった
- 結果: 重度障害者が遠隔で働けるロボットを開発

**Case C: 地域の高校生（モデルケース）**
- 強み: 「写真が上手い。地元の良さを誰より知っている」
- 自己認識: 「地元が好きでも発信の仕方を知らなかった」
- 結果: SNS発信で地域に観光客が増えた

# スライド9: グループワーク 概要（40〜60分）

## 「自分力マップ → アクション提言書」

**フェーズ1（15分）: 自分力マップ作成**
- 各自が「強み・価値観・経験」を書き出す
- マップをグループメンバーに紹介（1人2分）

**フェーズ2（15分）: グループ力の統合**
- メンバーの強みを合わせて「グループ力マトリクス」を作成
- 「このチームで何ができる？」を議論

**フェーズ3（20〜30分）: アクション提言書**
- 「この強みを使って、学校/地域/社会の課題を1つ解決するなら？」
- 課題・強みの活かし方・具体的なアクション・必要な資源を提言書に記載
