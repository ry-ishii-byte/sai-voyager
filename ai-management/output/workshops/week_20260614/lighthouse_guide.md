## ファシリテーションのポイント

### アイスブレイク（「わたし探しクイズ」）

- 読み上げる前に「誰の得意か想像しながら聴いてね」と伝えると集中力が上がる。
- 回収した紙をシャッフルして、書いた本人ではなく別の人が読み上げるバリエーションも可。
- 「こんな得意を持ってる人が隣にいたんだ！」という驚きがアイスブレイクの核心。
- よく出る反応とその対処：
  - 「得意なことなんてない」という子が書く前に言ったら：「『毎日学校に来ている』も得意なこと！毎日できることを書いていいよ」
  - 「友達の得意が全然わからなかった」場合：「初対面でも当てられるくらい、得意ってにじみ出るんだよ」とコメント

### スキル紹介（「自己認識と自己効力感」）

- 「自己効力感」は横文字で難しいが、「できる！って思える気持ち」と言い換えるだけで十分。
- スライド5の「4つの源泉」を全部説明するより、「①できた体験が一番大事」に絞って説明するほうが子どもには伝わりやすい。
- 航海メタファーを使う場合：「自分という船の特長を知ってるから、どの海に出るか決められる」

### ステップ1（得意×好き発掘）

- 「大したことない」が出やすいポイント。「謙遜禁止カード」という演出で「大したことじゃない！」と言いたくなる気持ちを笑いにかえる。
- 書けない子へのサポート例：
  - 「算数と国語、どっちが好き？」「体育は？図工は？」と教科別に聞く
  - 「休日に1人でいるとき、何してる？」
  - 「5年生の自分が3年生に教えてあげられることは？」
- チームメンバーが「それ得意そう！」と言ってあげる場面を作ると書きやすくなる。

### ステップ2（できたエピソード発掘）

- 「難しいこと」「大きなこと」じゃなくてよいと強調する。
  - 「ちょっと怖かった歯医者に1人で行けた」
  - 「はじめてのバイトで緊張したけど最後までできた」
  - 「苦手な人に『ありがとう』と言えた」
- 「やった後の自分をひとことで」の欄は、「達成感」「ほっとした」「もっとやれる気がした」などの感情語を引き出す。
  - ここで出てきた感情語がそのまま「自己効力感の体験」の記述になる。

### ステップ3（30秒スピーチチャレンジ）

**これがワーク最大のヤマ場。** 全員が30秒でも話せる環境を作ることが重要。

- 事前に「拍手の約束」を全員で確認する：「どんな内容でも、終わったら必ず拍手！」
- スピーチが苦手な子への声かけ：
  - 「30秒って短いよ。3文だけ言えばOK」
  - 「シートを読みながら話してもいい」
  - 「ひとことだけでも大丈夫！」
- 「やる前の自信度★」を必ず書かせてからスピーチさせる（後で比較するため）。
- 話し終わった直後が最も効果的なタイミング：「今どんな気分？スッキリした？」と問う。
- ほぼ全員が「終わったらスッキリした」「思ったよりできた」と感じる。その体験こそが本物の「自己効力感体験」。

### 全体発表・まとめ

- 板書に「やる前の★」と「やった後の★」の平均を書いて比べると視覚的に伝わりやすい。
- 「自信度が上がった」という数字が見えると、「今日ここで自己効力感が育った」という事実になる。
- 特定のよいスピーチを紹介する場合は、必ず本人の許可を取る。

### 巡回中のポイント

- ステップ1でペンが止まっているチームへ：「まずお互いの得意を教え合ってみて」と誘導。
- ステップ3前に緊張しているチームへ：「一番最初にやった人がチャンピオン！ガヤ不要」と声かけ。
- チームの雰囲気が暗いと感じたら：「チームが笑顔のうちはスピーチが上手くいく、よーいどん！」など元気を入れる。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| 「得意なことが全くない」と言う子がいる | 「毎日ご飯を食べられる」「昨日学校に来られた」も立派な得意！小さいことを探させる |
| スピーチを強く拒否する子がいる | 「シートを見せるだけでOK」「1文だけ読み上げるだけでOK」と段階的に妥協策を提示 |
| チームが笑ってスピーチを真剣に聴かない | 「笑いながら聴くのは最高の応援！でも茶化しはなし」と区別を伝える |
| 「得意」をチームに言うのが恥ずかしい | 「自慢じゃなくて自己紹介。知ってもらうことで助けてもらえる」と伝える |
| やる前もやった後も自信度が変わらない子がいる | 「変わらなかったのはなぜ？もともと自信があったから？」と深掘りする（それも立派な自己認識） |
| 時間が余ってしまう | アイスブレイクの「得意バトン」追加版（「○○が得意な人を探せ！」）で繋ぐ |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **自律力** | 自分の強み・価値観を知り、自分で自分を動かす源を発見する |
| **挑戦力** | 30秒スピーチという「小さな怖さ」に向き合い、「できた！」を体験する |
| 創造力 | 自分だけの「トリセツ」という形で自己を可視化・表現する |
| 協働力 | チームで互いの強みを発表し、讃え合いながら視野を広げる |
| 表現力 | 30秒という制約の中で、自分の言葉を選んで話す |
