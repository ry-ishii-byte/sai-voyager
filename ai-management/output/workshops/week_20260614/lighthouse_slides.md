# スライド2: アイスブレイク「わたし探しクイズ」（5〜10分）

## 隣の人の「得意なこと」を当ててみよう！

**ルール**

1. まず自分で「一番の得意なこと or 好きなこと」を紙に書く（秘密！1分）
2. 書いた紙を折ってファシリに渡す
3. ファシリが読み上げる → チームで「誰の得意？」を当てる！

**例**
> 「毎朝6時に起きて犬の散歩ができます」
> 「折り紙で鶴を目をつぶって折れます」
> 「嘘をついている人がすぐわかります」

✏️ **書く時間：1分。ウソはダメ！本当の自分を書こう。**

# スライド3: アイスブレイク 解説

## 「得意なこと」って、自分では当たり前なんだよね

> これが今日の大テーマ。

# スライド5: なぜ大事なの？（3分）

## 「できる！」という気持ちは、どこから来る？

**自己効力感を育てる4つの源泉**

```
① 「できた！」体験
     → 過去に成功した記憶が「また できる」という
       気持ちを作る

② 「あの人もできた！」
     → 自分と似た人が成功するのを見ると
       「わたしも！」と思える

③ 「あなたならできる！」
     → 信頼する人に言われると、
       自信が湧いてくる

④ 体の状態を整える
     → 深呼吸・姿勢・笑顔が
       「いける！」気持ちを作る
```

**今日は①「できた！体験」を自分の手で作る！**

    計算が早い、絵を描く、料理、ゲームの作戦を考える）
    → 思いつく限り書こう！最低3つ！

    ①
    ②
    ③
    ④ （あれば）

    →

# スライド8: ステップ2「できた！エピソード」（10分）

## シートのPart 2：チャレンジの記憶を呼び起こそう

**シートに書く：**

```
【Part 2: できた！エピソードを探せ】

Q1. 「難しそう」「無理かも」と思ったけど
    やってみたこと。どんなことだった？
    →

Q2. そのとき、どんな気持ちだった？（やる前）
    → ドキドキ / 不安 / やだな / その他（    ）

Q3. やってみた結果は？
    → できた！ / なんとかなった / 予想と違った

Q4. やった後、自分をひとことで表すと？
    →
```

💡 **小さな「できた！」でOK。宿題終わらせた、
　　はじめての電話、苦手な人に話しかけた——なんでもOK！**
