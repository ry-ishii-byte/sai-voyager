# 外部版 指導者ガイド
## テーマ: 不確実性・曖昧さ・リスクへの対処
**EntreComp領域**: 不確実性への対処  
**SAILの力**: 挑戦力  
**対象**: 小中学生（学校・自治体向け）  
**所要時間**: 120〜180分  
**推奨人数**: 20〜40名（4〜6名/チーム）  
**生成日**: 2026/05/03

## ファシリテーションの詳細ポイント

### ブロック① アイスブレイク（10分）

**狙い**: 「わからない」体験を言語化し、自分の反応パターンを自覚させる。

**進行手順**
1. A4用紙に書く時間は3分。「最近でなくてもOK。昔の体験でも」と伝える。
2. チーム内シェアは1人1分厳守（タイムキーパーを決めさせる）。
3. 「対処法の分類」は全員同意しなくてよい。「これは逃げた？探索した？」と議論させる。
4. タイプ分類（スライド4）を見せて「自分はどれ？」と聞く。多数決不要。一人ひとり違ってよい。

**注意点**
- 「フリーズ型」と答えた生徒を否定しない。「それを正直に言えることが、もう一歩目だよ」と伝える。
- チームに「わからない体験がない」と言う生徒がいたら「今日のワークで初体験できるよ」と笑顔で返す。

---

### ブロック② 導入講義（15〜20分）

**狙い**: 「不確実性・曖昧さ・リスク」の違いを理解し、なぜ今必要かを腹落ちさせる。

**問いかけタイミング（必須）**
- スライド5後：「不確実性と曖昧さ、似てるけど違う。どう違うと思う？」（30秒個人思考→1人指名）
- スライド6後：「テクノロジーが変わって、自分の生活で変わったことは？」（1分ペアトーク）
- スライド8後：「『失敗をデータとして使う』ってどういうこと？具体例ある？」（30秒→全体で1〜2人）

**時間オーバーを防ぐコツ**
- スライド6の「昔vs今」は読み上げせず「ここを見て30秒考えて」と投げる。
- スライド8の「3つの実践スキル」は黒板に書きながら説明するとテンポが出る。

---

### ブロック③ ケーススタディ（20〜30分）

**狙い**: AさんとBさんの違いを分析し、「小さく動く」の意味を「自分ごと」として捉える。

**議論ファシリのコツ**
- シナリオAのディスカッション（3分）後、チームの発言を1つずつホワイトボードに書く。
- 「他には？」と2〜3回聞いて引き出す。
- シナリオBを提示後「AとBの違いを具体的に3つ」を厳命する（「コミュニケーション」など抽象的な答えには「どのシーン？」と返す）。

**引き出したいキーワード（板書するもの）**
- 完璧を待たない（小さく動く）
- 行動がフィードバックを生む
- 「待つ」にもリスクがある
- 失敗 = 次の判断材料

---

### ブロック④ グループワーク（40〜60分）

**狙い**: 不確実な状況での計画立案を実体験する。「変化への対応」がメインの学習。

**「変化カード」配布のポイント**
- 20分経過後に静かに配布する（大げさに演出しなくてよい）。
- 「これが現実に近い。楽しんで」と一言添える。
- 180分版では2回配布（1回目：予算削減。2回目：さらに時間短縮など）。

**ヒントカード（詰まっているチームへ）**

```
困ってる？これ、使ってみて！

企画のアイデア例：
・朝のミニコンサート（楽器が得意な人が出演）
・教室ミニ写真展（スマホで撮った写真を貼る）
・クラスの「得意発表会」（5分ずつ特技を見せる）
・謎解きラリー（廊下に問題を貼る）
・みんなの好きなもの展（モノや写真を持ち寄る）

「小さくできること」から考えてみよう！
```

**巡回チェックリスト（1巡ごとに確認）**
1. 全員が付箋を書いているか？
2. 「リスクと対処法」欄に取り組んでいるか？
3. 「変化カード」を受け取ったあと、修正が始まっているか？
4. 残り時間を把握しているか？

**詰まっているチームへの問いかけ例**
- 企画が決まらない → 「一番小さく始められるものはどれ？」
- リスクが書けない → 「もしこの企画がうまくいかないとしたら、何が原因？」
- 変化カードに反応できない → 「予算が半分になったとき、何を諦めて、何を残す？」

---

### ブロック⑤ プレゼンテーション（20〜30分）

**狙い**: 「どう動いたか（プロセス）」を言語化する。変化への対応の違いを学び合う。

**フィードバックカードのルール再確認**
- 💛（いいね）は1枚以上書く（必須）
- 💙（改善案）は「○○したらもっと良くなる」という言い方で書く（批判NG）

**ファシリの介入ポイント**
- 「企画の内容」よりも「どう変化に対応したか」を深掘りする問い返しをする。
- 発表が3分を超えたらやさしく「あと30秒でまとめよう」と声かけ。
- 全チーム終了後「変化カードを受け取ったとき、一番面白かった対応は？」を全体で聞く。

---

### ブロック⑥ 振り返り・自己評価（15〜20分）

**狙い**: 体験を「学び」に変える。「挑戦力」の言語化とメタ認知を育てる。

**進行**
1. 振り返りシートを配布（または記入済み箇所がある場合はその続き）。
2. 「静かに自分で書く時間」を10〜15分確保。急かさない。
3. 180分版の場合：2人ペアで読み合い、感想を1つ伝え合う（3分）。

**書けない生徒へのヒント問いかけ**
- 問い1が書けない → 「変化カードが来たとき、どんな気持ちだった？」
- 問い3が書けない → 「今日、自分が一番勇気を出した瞬間はいつ？」
- 問い4が書けない → 「明日、学校で一つ『とりあえずやってみること』は何？」

## SAILの5つの力との接続（詳細）

| SAILの力 | このワークでの体験 | 引き出す問い |
|----------|-----------------|------------|
| **挑戦力** | 変化という「予測不能な状況」に飛び込む体験 | 「変化カードが来たとき、どう感じた？」 |
| 自律力 | 他者の答えを待たず、自分たちで判断・修正する | 「誰にも聞かずに決めた場面は？」 |
| 創造力 | 制約が増えた中でも新たな解決策を発明する | 「制約があったから生まれたアイデアは？」 |
| 協働力 | 不確かな状況でチームで判断・行動を合わせる | 「チームだから乗り越えられたことは？」 |
| 表現力 | 「なぜそう決めたか」をプレゼンで言語化する | 「変化への対応をどう説明した？」 |

## SAIL内部メモ

**このワーク設計の意図**
- 「変化カード」による制約追加は、「不確実性」を教室で安全に体験させる最重要仕掛け。配布タイミングは早すぎず（20分目）、遅すぎず（30分以降は修正時間が不足）が基本。
- ケーススタディのAさんは「完璧主義による先送り」を体現。これは多くの生徒・教員が共感しやすいパターンのため、自己開示につながりやすい。
- 企画テーマを「学校祭」にしたのは、生徒が自分ごとして捉えやすいため。地域・学校によって「クラス会」「部活イベント」「委員会企画」などにカスタマイズ可能。
- 「リスクと対処法」欄を企画書に必須としているのは、EntreCompの「リスク評価」要素を自然に組み込むため。
