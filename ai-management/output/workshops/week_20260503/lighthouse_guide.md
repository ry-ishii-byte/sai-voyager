## ファシリテーションのポイント

### アイスブレイク

- 謎の絵カードは「正解があるもの」を使わない。異国の道具や抽象的な形がベスト。
- 「変な答えほど面白い！」と明示して、心理的安全性を高める。
- ABCの手挙げで「C（動けなくなる）」を挙げた子を笑わない。「正直に言ってくれてありがとう！」と必ず受け止める。

### スキル紹介

- スライド5の表は読み上げるより「どっちが好き？」と聞いてから見せると効果的。
- 「霧の中でも帆を張れ」は航海メタファー。「霧 = わからないこと」「帆 = とりあえずやること」と補足すると小学生にも伝わる。

### メインワーク（タワーチャレンジ）の核心

**「意図的な不確実性」を体験させることが目的。**

- ワーク説明時、意図的に「何を作るか言わない」「ルールが追加される」ことを予告しない。
- 5分経過時のルール変更発表は、落ち着いた声で静かに行う（わざと大げさにしない）。
- 「えーっ！」という反応が出たら「こういうことが実際の仕事でも起きる」と一言添える。

### 巡回中のポイント

- 手が止まっているチームへ: 「今、何に迷ってる？」と聞いて、判断を促す。
- ルール変更後に混乱しているチームへ: 「手元にあるものを全部並べてみよう」と整理を促す。
- 一人が仕切りすぎているチームへ: 「○○さんはどうする？」と他のメンバーに振る。

### 発表・計測

- 高さだけでなく「どうやって決断したか」を一言聞く。「何に困って、どう解決した？」
- チームによって戦略が違うことを強調する。「みんな同じ材料なのに、こんなに違う！」

### 振り返りシート

- 「書けない」子には「今日一番ドキドキした瞬間はいつ？」と聞いてから書かせる。
- 「★いくつにした？」を全体で聞くと場が温まり、書き終えた子も参加できる。

## よくある問題と対処法

| 問題 | 対処 |
|------|------|
| 「何を作ればいいの？」としつこく聞かれる | 「今はまだ秘密！とりあえず材料を眺めてみよう」とかわす |
| 最初から高い目標を立てて作りすぎる | ルール変更後に「あ、これは経験だ」と笑顔で受け止めさせる |
| ルール変更でパニックになるチームがある | 「残った材料を机に並べて。まず何本残ってる？」と落ち着かせる |
| タワーがすぐ倒れて意欲が下がる | 「倒れ方から学ぶ。何が弱かった？」と問いかけ、再挑戦を促す |
| 振り返りシートが白紙のまま終わる | ★だけ先につけてもらい「なんで★3にした？」と引き出す |

## SAILの5つの力との接続

| SAILの力 | このワークでの体験 |
|----------|------------------|
| **挑戦力** | ルール変更という「予測できない変化」に飛び込む体験 |
| 創造力 | 制約が増えた中で新たな解決策を発見する |
| 自律力 | 「誰かに答えを聞く」ではなく、自分たちで決断する |
| 協働力 | 混乱した状況でチームとして判断・行動する |
| 表現力 | 「どう対処したか」を発表で言語化する |
