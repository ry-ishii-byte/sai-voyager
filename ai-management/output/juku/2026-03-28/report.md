## STEP 1: 時期分析

### 現在の状況
- **日付**: 2026年3月28日（土）
- **季節**: 年度末・春休み突入直前

### 近いイベント・節目
| 時期 | イベント |
|------|---------|
| 3月末 | 公立高校・中学受験の結果発表ほぼ完了 |
| 3月下旬〜4月上旬 | 春休み（約2週間） |
| 4月上旬 | 新学期スタート・入学式 |
| 4月中旬 | 新入生オリエンテーション期間 |

### この時期に保護者・生徒が抱える心理

**【小学生保護者】**
- 「来年度こそは勉強習慣をつけさせたい」
- 「中学入学前に算数・国語の基礎を固めておきたい」
- 「新しい環境で子どもが不安そう…どう支えれば？」

**【中学生・保護者】**
- 「新学年になって最初のテストで失敗したくない」
- 「高校受験まであと1〜2年、今から何をすべきか」
- 「春休み中に苦手科目を克服したい」

**【受験生（合格組）保護者】**
- 「合格できた！でも入学後についていけるか心配」
- 「入学準備として何か勉強しておくべき？」

### 施策の方向性
この時期は「新しいスタートへの期待と不安」が入り混じるタイミング。
「入塾の売り込み」ではなく、**「一緒に新年度を迎えよう」という伴走姿勢**を示すことが最も効果的。
すでに通っている生徒の「達成感」を可視化し、それを見た新規層が「うちの子もここで成長できそう」と感じる流れを作る。

## STEP 2: 施策一覧（5本）

---

### 施策① 「はじめの一歩」無料体験授業 × 新学年応援メッセージカード
**優先度: 高 | コスト: ほぼゼロ**

| 項目 | 内容 |
|------|------|
| 目的 | 春の入塾ピークに合わせた新規体験者獲得 |
| ターゲット | 新中1・新高1・学年が上がる小4〜中2の保護者・生徒 |
| 手法 | 体験授業参加者全員に講師の手書きメッセージカードを渡す。「〇〇さんはこの教科が得意になると思います」など個別に一言添える。SNS・LINE・塾の前貼り紙で告知。 |
| コスト | 紙・ペン代のみ（実質ゼロ）。スタッフの誠意で付加価値を作る。 |
| 関係性への効果 | 初めて来た生徒・保護者に「見てもらえた」という安心感を与え、信頼の出発点を作る。カードは家に持ち帰られ保護者の目にも届く。 |
| 実施タイミング | 今週中（3月28日〜31日に体制整備・告知開始、4月1日〜体験受付） |

### 施策② 卒業生・先輩から「中学・高校の本音」を聞く会
**優先度: 高 | コスト: ほぼゼロ**

| 項目 | 内容 |
|------|------|
| 目的 | 新中1・新高1の不安を解消し、入塾動機と塾へのロイヤリティを高める |
| ターゲット | 新中1（小6卒業生とその保護者）、新高1（中3卒業生とその保護者） |
| 手法 | 塾の卒業生（1〜2年先輩）に「進学後のリアル」を話してもらう30分イベントを塾内で開催。「中学は何が難しかった？」「部活と勉強の両立は？」など保護者・生徒が聞きたいことを事前収集してQ&A形式で。 |
| コスト | ほぼゼロ（卒業生へのお礼は感謝状や手書きカードで十分） |
| 関係性への効果 | 卒業生との絆継続、在塾生・新入生の不安解消、保護者の信頼向上。「塾に来ると仲間がいる」という安心感が生まれる。 |
| 実施タイミング | 2週間以内（4月上旬に開催できるよう3月中に卒業生へ声がけ） |

### 施策③ 保護者向け「春の勉強習慣づくり」ミニセミナー（無料・30分）
**優先度: 高 | コスト: ほぼゼロ**

| 項目 | 内容 |
|------|------|
| 目的 | 保護者との信頼関係構築と口コミ促進 |
| ターゲット | 在塾生の保護者 + 体験希望者の保護者 |
| 手法 | 「春休み中に家で子どもの勉強習慣を作る5つのコツ」をテーマに塾長・講師が30分で話すミニセミナー。Zoom or 塾内で開催。参加者が口コミしたくなるような実用的な内容にする。終了後に感想カードを回収してフィードバックを得る。 |
| コスト | ほぼゼロ（Zoom無料プラン or 塾内スペース利用） |
| 関係性への効果 | 保護者が「この塾は売り込まない、本当に子どものことを考えている」と感じる。参加した保護者同士のコミュニティも生まれやすい。 |
| 実施タイミング | 今月中（3月30日〜4月5日の週末 or 平日夜に開催） |

### 施策④ 「春休みチャレンジ宣言」ボード（塾内掲示 + SNS発信）
**優先度: 中 | コスト: ほぼゼロ**

| 項目 | 内容 |
|------|------|
| 目的 | 在塾生のモチベーション維持 + 塾の雰囲気をSNSで発信し認知拡大 |
| ターゲット | 在塾生（主体）、SNSを見る保護者・見込み客 |
| 手法 | 塾の入口や掲示板に大きなボードを設置。生徒が「春休みにやること宣言」を付箋に書いて貼る。写真をInstagramに投稿（生徒の顔は写さない）。新学期に「達成できた？」とフォロー。 |
| コスト | 画用紙・付箋代のみ（100〜200円） |
| 関係性への効果 | 生徒が「塾に来たくなる」仕掛けになり、通塾習慣が定着。保護者・見込み客には「活気ある塾」の雰囲気が伝わる。 |
| 実施タイミング | 今週中（3月28日〜29日に設置） |

### 施策⑤ 地域の小学校・中学校向け「進学準備お役立ちプリント」配布提案
**優先度: 中 | コスト: 少額（印刷費のみ）**

| 項目 | 内容 |
|------|------|
| 目的 | 地域への貢献と認知度向上、新規問い合わせの獲得 |
| ターゲット | 近隣小学校・中学校の先生 → 児童・生徒・保護者へ |
| 手法 | 「中学校入学前にやっておきたい算数・英語の基礎チェックリスト」など役立つ学習プリントを作成し、学校の先生を通じて配布許可を得て配布。塾名・連絡先を小さく記載。押し売り感を出さず「地域の子どもたちへのプレゼント」として提供。 |
| コスト | 少額（A4印刷代、50〜100枚程度で数百円） |
| 関係性への効果 | 学校の先生との信頼関係構築。「地域のために動く塾」として保護者からの印象が向上。 |
| 実施タイミング | 今月中（4月の新学期に間に合うよう3月中に学校へ連絡） |
