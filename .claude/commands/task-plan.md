# タスク計画をDiscordに送信

以下の手順でタスクとカレンダーを照合し、担当者ごとの計画を作成してDiscordに送信してください。

## Discord メンション情報

- 石井: `<@1105463409371267092>`
- 内藤: `<@1377953149947744378>`

## Step 1: スプレッドシートから未着手タスクを取得

以下のURLからCSVを取得し、**未着手または作業中**のタスクを担当者別に抽出してください。

- スプレッドシートURL（gid=1005158658）:
  `https://docs.google.com/spreadsheets/d/1GjCpRj6nOHXkAwovaYlTozDzw2GZ23sQuAnZGxP3_Xc/export?format=csv&gid=1005158658`

抽出条件: 作業状態列 = 「未着手」または「作業中」

担当者ごとに分類:
- 石井担当タスク
- 内藤担当タスク

## Step 2: 今後7日間のカレンダーを取得

Google Calendar MCPを使い、今日から7日間の予定を取得してください。
`timeMin`: 今日の00:00:00+09:00
`timeMax`: 7日後の23:59:59+09:00

## Step 3: 空き時間を特定し、タスクを配置

カレンダーの予定と未着手タスクを照合し、以下のルールで配置してください：

- 期限切れのタスクは最優先で今日に配置
- 期限が近いものを優先
- 同じ性質のタスク（連絡・発注・作業など）はまとめて同じ時間帯に配置
- 移動時間・授業・MTGの隙間を活用

## Step 4: Discordに送信

以下のcurlコマンドで計画をDiscordに送信してください。
`$DISCORD_WEBHOOK_URL` 環境変数を使用します。

```bash
curl -X POST "$DISCORD_WEBHOOK_URL" \
  -H "Content-Type: application/json" \
  --data-binary @- <<'PAYLOAD'
{
  "content": "<ここに計画テキストを入れる>"
}
PAYLOAD
```
