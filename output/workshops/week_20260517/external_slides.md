# スライド5: 導入講義② — 機会と価値

# 「機会」と「価値」の正体

**機会とは？**
> 誰かが「困っている・不便・もったいない」と感じている場所に潜む、解決できるチャンス

**価値とは？**
> その解決によって生まれる「誰かの嬉しさ・助かり・笑顔」

---

**変換の公式**

```
困りごと → 「なぜ？」 → 本質的な問い → アイデア → 誰かの笑顔（価値）
```

---

# スライド6: 導入講義③ — なぜ？を3回掘る

# 「なぜ？」を3回掘ると根っこが見える

**例：「電車でスマホを落とす人が多い」**

- なぜ？ → 手が滑るから
- なぜ？ → 画面が大きくて持ちにくいから
- なぜ？ → スマホを片手で操作しようとするから

**本質の問い**：「片手で安全に操作できるスマホ体験がない」

→ ここにアイデアが生まれる

---

# スライド7: ケーススタディ説明

# 先輩たちのケースを見てみよう（20〜30分）

**3つのケースから1つ選んで読む**

| ケース | 概要 |
|---|---|
| A：通学路の危険 | 小学生が危険な通学路を地図化してアプリにした中学生 |
| B：食品ロス | 給食の残り物削減を呼びかけた高校生の取り組み |
| C：外国人の困りごと | 観光客の言語バリアに気づき多言語案内をつくった中学生 |

> **資料はグループに配布済み**

# スライド8: ケーススタディ 討議

# ケースを読んだら、グループで話し合おう

**3つの問い**
1. この人たちは、どんな「機会（困りごと）」に気づいたか？
2. 彼らが生み出した「価値」は何か？
3. あなたのグループなら、どんな別のアイデアがあるか？

**時間：15分 → 全体共有5分**

# スライド9: グループワーク説明

**ゴール**：日常の困りごとから「機会」を見つけ、

**使うツール**：価値の設計図シート（配布済み）

---

# スライド10: 価値の設計図シートの使い方

# 設計図シートの記入手順

**STEP 1（10分）**: 困りごとの洗い出し
- グループで「困った・不便・もったいない」を付箋に出す
- 対象：自分・家族・地域・学校など

**STEP 2（10分）**: 困りごとを深掘り
- 1つ選んで「なぜ？」を3回繰り返す

**STEP 3（15分）**: アイデアと価値を設計
- 解決アイデアを複数出す
- 「誰が」「どんなふうに嬉しくなるか」を言語化

**STEP 4（10分）**: 発表準備
- 設計図を整理し、チームで発表の流れを決める
