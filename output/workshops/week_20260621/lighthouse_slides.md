# LIGHTHOUSE ワークショップ スライド原稿
## テーマ：自己認識と自己効力感
**対象**: 小中学生 ／ **所要時間**: 60〜90分 ／ **形式**: 集客イベント

---

# スライド1: オープニング

# ようこそ、SAIL LIGHTHOUSE へ！⛵

**今日のテーマ**
「自分のこと、どれだけ知ってる？」

> ファシリテーター用メモ: 「今日は"自分探検"をしてもらうよ！見つけたものにびっくりするかも！」と笑顔でスタート。

---

# スライド2: アイスブレイク

# まず、自分をのぞいてみよう！（5〜10分）

**「わたし自慢ベスト3」**

紙に書いてみよう！どんな小さいことでもOK：

- ⭐ 得意なこと・うまいこと
- 😊 好きなこと・ハマっていること
- 🏅 「これだけは負けない！」と思うこと

**2分間で3つ、書いてみよう！**

> ファシリ：「勉強・スポーツじゃなくていいよ。『お母さんより速く服をたためる』も立派な自慢！」

---

# スライド3: アイスブレイク 全体共有

# 自慢、聞かせて！

**2〜3人に発表してもらおう**

> ファシリ：「おお〜！」「それすごい！」と全力でリアクション。小さい自慢ほど大げさに褒める。
> 「意外な才能を持つ人がいっぱいいるね！」と全体にフィードバック。

---

# スライド4: 今日のスキル紹介

# 今日学ぶ力：**自己認識と自己効力感**

**EntreComp スキル⑤**
領域：資源の動員 ／ 力：自律力・挑戦力

---

**2つの言葉、聞いたことある？**

**自己認識** ＝ 「自分のこと（強み・弱み・好き・苦手）をよく知っている」こと

**自己効力感** ＝ 「自分ならできる！」という気持ちの強さ

> 難しそう…？でも大丈夫。今日のワークで体験しながら学ぶよ！

---

# スライド5: なぜ大切？

# 「自分を知る」ってなぜ大事なの？

**例えば…**

サッカーが好きな中学生 → 「俺、走るのは得意でも、シュートが苦手」と知っている
→ 「じゃあシュート練習を増やそう！」→ どんどん上手くなる⚽

---

**自分を知らないと…**
- 得意なことに気づかないまま
- 「どうせ無理」と思って挑戦できない

**自分を知ると…**
- 強みをもっと活かせる
- 「苦手」もチャレンジに変えられる

---

# スライド6: メインワーク説明

# ワーク：「わたし取扱説明書」をつくろう！

**ルール**
- 個人 → ペアでシェア → グループで深掘りの順に進む
- 制限時間：**35分**
- 取扱説明書シートを使う（配布済み）

**ゴール**：自分だけの「取扱説明書」を完成させて、自分の強みを3つ発見する！

---

# スライド7: ワーク STEP 1

# STEP 1：「強みカード」から選ぼう（10分）

**やること**
テーブルの上の「強みカード」を全部見て、自分に当てはまると思うものを**3〜5枚**選ぶ

**カードの例**
- 🧐 よく観察する　　📣 みんなをまとめる
- 🎨 アイデアが出る　　💪 あきらめない
- 🤝 話を聞くのが上手　　📚 調べるのが好き
- 😄 笑わせるのが得意　　⚡ 行動が速い

> **目標：自分の「好き」と「得意」から選ぼう。正直に！**

---

# スライド8: ワーク STEP 2

# STEP 2：「わたし取扱説明書」を書こう！（15分）

**取扱説明書シートに記入！**

| 項目 | 書くこと |
|---|---|
| 🏷 モデル名（あだ名） | 今日だけの自分の名前をつけよう！ |
| ⭐ 最大の強み TOP3 | 選んだカードから3つ書く |
| 🔋 エネルギーが出るとき | こんなときやる気が出る！ |
| ⚠️ ちょっと苦手なこと | 正直に1つ書いてみよう |
| 💡 できるようになりたいこと | 1つだけ |

> **ペアで見せ合って「確かに！」「それ知らなかった！」とコメントし合おう！**

---

# スライド9: ワーク STEP 3

# STEP 3：「できた！」体験を思い出そう（10分）

**過去の自分を振り返ろう！**

「むずかしかったけど、できた！」
「はじめは怖かったけど、やってみたら楽しかった！」

**そのときのことを書いてみよう**

| 質問 | あなたの答え |
|---|---|
| どんなことができた？ | |
| そのときどんな気持ちだった？ | |
| 何が自分を助けてくれた？ | |

> **「自己効力感」はこの「できた体験」から育つよ！**

---

# スライド10: 発表タイム

# 「わたし」を紹介しよう！（10〜15分）

**グループ内で発表（1人30秒〜1分）**

1. 自分のモデル名
2. 最大の強み TOP3
3. 「できた！」体験で一番自慢したいこと

> 聞いているみんなへ：「そこが強みなの、わかる！」と思ったらうなずこう！
> ファシリ：発表後に「そのエピソード、すごい！なんでそこで頑張れた？」と深掘りする。

---
