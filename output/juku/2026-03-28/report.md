## 時期分析

### 現在の状況
- **時期:** 3月下旬（新年度前最後の週）
- **学校カレンダー上の位置:** 公立高校入試合格発表ほぼ完了・卒業式シーズン・春休み突入直前

### 保護者・生徒の心理状態

| 対象 | 不安・期待 |
|------|-----------|
| 新中1予定の小6 | 中学の勉強量・部活との両立・友達づくりへの不安 |
| 新高1予定の中3 | 高校数学・英語の急激な難化への恐れ |
| 継続在塾生 | 新学年で気持ちを切り替えたい・良いスタートを切りたい |
| 保護者全般 | 「今の塾のまま大丈夫か」「新しいサポートが必要では」という再評価期 |

### 集客機会ポイント
1. **卒業生の合格報告**が口コミの最大の起爆剤になる時期
2. 入塾を迷っていた家庭が「4月から」を決断するラストチャンス
3. 在塾生がリニューアルした気持ちで塾を続けるかどうかの分岐点

## 施策一覧（5本）

### 施策① ★ 卒業生「合格体験談シェア」ミニイベント
| 項目 | 内容 |
|------|------|
| **目的** | 信頼構築・強力な口コミ創出・新規入塾の後押し |
| **ターゲット** | 新中1・新高1予定者とその保護者 |
| **手法** | 今年度入試合格の卒業生2〜3名に「どうやって勉強したか」を話してもらう30分の座談会。塾内で無料開催。保護者も参加可。来られない人向けにコメントを紙にまとめて配布 |
| **コスト** | ほぼゼロ（お菓子代のみ） |
| **関係性への効果** | 卒業生の「恩返し」意識が育つ。後輩との縦のつながりが生まれる |
| **実施タイミング** | 4月1週目（新学期直前） |
| **優先度** | 🔴 高 |

### 施策② ★ 「新学期スタートダッシュ準備チェックシート」無料配布
| 項目 | 内容 |
|------|------|
| **目的** | 見込み顧客との自然な接点づくり・塾の専門性アピール |
| **ターゲット** | 全学年の保護者・生徒（特に進学予定者） |
| **手法** | 「新中1が最初の1ヶ月でやるべき5つのこと」「新高1が絶対にやってはいけない勉強法」などの実用チェックシートをA4一枚にまとめ、SNS・LINEで無料配布。塾名・連絡先を入れるが売り込み文なし |
| **コスト** | ほぼゼロ（制作：スタッフの時間のみ） |
| **関係性への効果** | 「役に立つ情報をくれる塾」として認知。保護者の信頼を事前に獲得 |
| **実施タイミング** | 今週中（3月28日〜4月3日） |
| **優先度** | 🔴 高 |

### 施策③ 保護者向け「中学・高校 学習の落とし穴」無料説明会
| 項目 | 内容 |
|------|------|
| **目的** | 保護者との直接関係構築・入塾検討のきっかけ |
| **ターゲット** | 新中1・新高1の保護者（既存・見込み問わず） |
| **手法** | 「中学で成績が下がる本当の理由」「高校で挫折する生徒のパターン」を30分で解説する無料説明会。売り込み一切なし。最後に質問タイムあり。知人を誘ってきてもらうよう既存保護者にお願い |
| **コスト** | ほぼゼロ |
| **関係性への効果** | 「この塾の先生は信頼できる」という感情的な結びつきが生まれる |
| **実施タイミング** | 4月1〜2週目の土曜日 |
| **優先度** | 🟡 中 |

### 施策④ 在塾生全員への手書き「春の応援カード」
| 項目 | 内容 |
|------|------|
| **目的** | 在塾生の継続率向上・家庭への口コミ促進 |
| **ターゲット** | 全在塾生 |
| **手法** | 担当講師が一人ひとりに手書きのひと言メッセージカードを渡す。「〇〇さんが△△を頑張っていたこと、先生は見ていました」など個人に向けた内容にする |
| **コスト** | ほぼゼロ（カード代のみ） |
| **関係性への効果** | 生徒・保護者との強い情緒的つながり。「この塾に通い続けたい」という気持ちが強まる |
| **実施タイミング** | 3月31日〜4月4日 |
| **優先度** | 🟡 中 |

### 施策⑤ 春休み「無料自習室開放デー」
| 項目 | 内容 |
|------|------|
| **目的** | 地域認知向上・未体験者への体験機会提供 |
| **ターゲット** | 近隣の小中学生（塾未経験者も歓迎） |
| **手法** | 春休み中の1〜2日間（10時〜16時）、塾の自習室を完全無料開放。予約不要で誰でも来られる。スタッフが勉強の質問に無料対応。SNS・近隣へのポスティングで告知 |
| **コスト** | 少額（スタッフの人件費相当のみ） |
| **関係性への効果** | 「敷居なく通える塾」としての地域イメージが定着。初めての来塾体験が生まれる |
| **実施タイミング** | 春休み期間中（3月30日〜4月5日のうち1〜2日） |
| **優先度** | 🟢 低〜中 |
