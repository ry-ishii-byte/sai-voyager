## 🎯 今週の集客施策（5本）

### 施策① 【優先度：高】新学年スタートダッシュ無料体験授業
| 項目 | 内容 |
|---|---|
| **目的** | 春休み中に体験→4月入塾の流れを作る |
| **ターゲット** | 新中学1年生・新高校1年生・学年が上がる在校生 |
| **手法** | 春休み期間中（3/28〜4/6）に1〜2回の無料体験授業を開催。「新学年の予習ができる」内容にして保護者にも訴求。既存生徒に友人紹介を依頼し、紹介者・被紹介者双方に手書きカードを贈る |
| **コスト** | ほぼゼロ（既存の授業枠を活用） |
| **関係性への効果** | 体験生が「塾の雰囲気」を実感できる。既存生徒の紹介により友人関係を塾に取り込める |
| **実施タイミング** | 今すぐ〜4月6日 |

### 施策② 【優先度：高】卒業生から後輩へ「合格の秘訣」手紙プロジェクト
| 項目 | 内容 |
|---|---|
| **目的** | 合格実績のリアルな声を広め、塾への信頼を高める |
| **ターゲット** | 在塾生・体験生・その保護者 |
| **手法** | 今年度の卒業生（高校・大学合格者）に手書きまたはデジタルメッセージを依頼。「勉強で一番きつかったこと」「この塾で変わったこと」をA5サイズ1枚にまとめてもらう。教室に掲示・体験授業時に配布・SNSに投稿（許可取り） |
| **コスト** | ほぼゼロ（用紙代のみ） |
| **関係性への効果** | 卒業生が「塾の一部」として関わり続けられる。在塾生に「先輩がいる」という安心感。見込み生徒に最強のリアル実績 |
| **実施タイミング** | 3月中に依頼・4月第1週に掲示・配布 |

### 施策③ 【優先度：高】保護者向け「新学年 勉強の始め方」ミニ相談会
| 項目 | 内容 |
|---|---|
| **目的** | 塾を「相談できる場所」として認知させ、関係構築〜入塾へ |
| **ターゲット** | 塾未加入の小中学生の保護者 |
| **手法** | 4月第1〜2週の平日夜（19〜20時）に30分のミニ相談会を開催（定員5〜8組）。「学年別・新学期から成績を上げる家庭学習のポイント」をテーマに講師が話す。個別相談タイム付き。チラシ不要、LINEと口コミのみで告知 |
| **コスト** | ほぼゼロ（会場は塾の教室） |
| **関係性への効果** | 保護者と講師が直接話すことで「この先生なら安心」という信頼が生まれる |
| **実施タイミング** | 4月1日〜15日 |
