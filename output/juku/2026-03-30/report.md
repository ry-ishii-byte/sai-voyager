### 保護者・生徒が抱える「不安」と「期待」
**不安:**
- 新学年で勉強についていけるか（特に新中1・新高1）
- 受験に向けてどこから手をつければいいか
- 入塾時期を逃すのではないか

**期待:**
- 新学年で心機一転したい
- 春から良いスタートを切りたい
- 合格できた喜び・次のステップへの意欲（受験組）

### 集客機会の要点

## STEP 2: 集客施策 一覧（5本）

### ① 進級・合格おめでとう手書きカード手渡し
| 項目 | 内容 |
|-----|------|
| **目的** | 既存生徒・保護者との絆を深め、口コミ・紹介を促進する |
| **ターゲット** | 在塾生全員・特に受験を終えた中3・高3 |
| **手法** | 講師・スタッフ全員で生徒一人ひとりに手書きメッセージカードを作成し手渡し。「今年も頑張ったね」「来年度もよろしく」など個人に向けたメッセージを添える。保護者には別途お礼のカードを封筒に入れて渡す |
| **コスト** | ほぼゼロ（100均カード・手書き） |
| **関係性への効果** | 「大切にされている」実感→継続率向上・自然な口コミ発生 |
| **実施タイミング** | 3月30日〜4月10日（春期講習・新学期直前） |
| **優先度** | 🔴 高 |

### ② 卒業生合格体験談LINEレポート配信
| 項目 | 内容 |
|-----|------|
| **目的** | 新中3・高3の保護者と受験未経験層への信頼醸成・入塾検討促進 |
| **ターゲット** | 在塾生保護者（LINE）・Instagram/X フォロワー |
| **手法** | 今春合格した卒業生5〜10名の「合格体験談」を短文でまとめ、LINEとSNSで配信。「どう勉強したか」「塾で何が役立ったか」を本人の言葉で。卒業生の承諾を得て実名（ニックネーム可）・志望校名・一言コメントを掲載 |
| **コスト** | ほぼゼロ（卒業生への依頼・文章まとめのみ） |
| **関係性への効果** | 卒業生との継続的なつながり・在塾生へのモチベーション提供 |
| **実施タイミング** | 3月31日〜4月5日 |
| **優先度** | 🔴 高 |

### ③ 「新中1・小6保護者向け 無料相談DAY」開催
| 項目 | 内容 |
|-----|------|
| **目的** | 新中1（現小6）保護者の不安を解消し、自然な入塾につなげる |
| **ターゲット** | 地域の小6保護者・現小5保護者 |
| **手法** | 「中学入学前にやっておくべきこと」をテーマに15〜20分の個別相談会を開催。予約制でLINE・電話受付。「売り込みなし・情報提供のみ」をポスターに明記。相談後に体験授業の案内を渡す（押しつけなし） |
| **コスト** | ほぼゼロ（スタッフの時間のみ） |
| **関係性への効果** | 「親身な塾」として地域での認知確立。相談→入塾の自然な流れ |
| **実施タイミング** | 4月4日（土）・5日（日）開催、3月末から告知 |
| **優先度** | 🔴 高 |
