# LIGHTHOUSE 新拠点開設 準備物リスト

> 作成日：2026-03-24
> 用途：実践型起業ラボ「LIGHTHOUSE」の新拠点を開設する際の準備チェックリスト

---

## 1. 物件・設備

- [ ] 物件契約（賃貸借契約書）
- [ ] 什器・家具（机、椅子、ホワイトボード等）
- [ ] 清掃用品

---

## 1-A. 機材・デジタル環境

### ネットワーク
- [ ] 光回線またはNURO等の高速インターネット契約
- [ ] Wi-Fiルーター（生徒10名以上が同時接続できる性能のもの）
  - 推奨：Wi-Fi 6対応、2.4GHz/5GHz デュアルバンド
- [ ] 有線LANポートの確保（プロジェクター・スタッフPC用）
- [ ] ネットワーク機器の設置場所・配線の確認

### 生徒用デバイス
- [ ] タブレット（人数分＋予備2台）
  - OS：iPad（推奨）または Android
  - ケース・フィルムの装着
  - 充電ケーブル・アダプターの人数分確保
- [ ] PC（可能であれば。Chromebook等でも可）
  - キーボード操作に慣れさせる場合は必須
- [ ] デバイス管理用Googleアカウント（拠点共用）の作成
  - 生徒が個人アカウントを使わなくて済む仕組みの整備

### スタッフ用デバイス
- [ ] スタッフ用ノートPC（1台以上）
  - Googleスライド・スプレッドシート等が快適に動作するスペック
- [ ] スタッフ用スマートフォン（撮影・Comiru通知確認用）

### 映像・音響
- [ ] プロジェクターまたは大型ディスプレイ（50インチ以上推奨）
  - プロジェクターの場合：スクリーンまたは投影可能な白壁の確認
- [ ] HDMIケーブル（複数本）
- [ ] HDMIセレクター（PC・タブレット・スタッフPCを切り替えたい場合）
- [ ] Type-C → HDMI 変換アダプター（iPad・MacBook対応）
- [ ] Bluetoothスピーカーまたは小型スピーカー（BGM・動画再生用）
- [ ] マイク（ワークショップ登壇・プレゼン発表時の補助用）

### 周辺機器・その他
- [ ] プリンター（印刷・スキャン対応）
  - ワークシート・申込書の現地印刷用
  - インクカートリッジの予備を複数確保
- [ ] 延長コード・電源タップ（タブレット充電ステーション用）
  - 口数：（生徒用デバイス数 ÷ 2）本以上を目安
- [ ] USBハブ（データ共有・充電補助用）
- [ ] Webカメラ（オンライン配信・保護者向け公開授業対応の場合）

### 初期設定チェック
- [ ] 全タブレット・PCのWi-Fi接続確認
- [ ] プロジェクター／ディスプレイとの接続テスト
- [ ] Googleスライド・Jamboard等の授業ツールの動作確認
- [ ] Comiruアプリのインストール・ログイン確認（スタッフ端末）
- [ ] 充電ケーブルの長さ・配置の確認（授業中に邪魔にならないか）
